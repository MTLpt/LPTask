# ☁️ LPTask (Sky To-Do List & Task Manager)

**LPTask** คือเว็บแอปพลิเคชันและเดสก์ท็อปแอปพลิเคชันสำหรับจัดการงานและตารางเวลา ที่ออกแบบมาในธีมท้องฟ้า ใช้งานง่าย และสามารถบันทึกข้อมูลไว้ในเครื่องได้โดยไม่ต้องใช้ฐานข้อมูลภายนอก

## ✨ ฟีเจอร์หลัก (Features)

- 📝 **จัดการรายการงาน (Task Management):** เพิ่ม, แก้ไข, ลบ และตั้งค่าความสำคัญของงาน (🔴 มาก, 🟡 ปกติ, 🟢 น้อย)
- 📅 **ปฏิทินงาน (Calendar View):** ดูภาพรวมงานของคุณในรูปแบบปฏิทินรายเดือน และสามารถคลิกดูงานในแต่ละวันได้
- 🏷️ **จัดการหมวดหมู่ (Categories):** สร้างและกำหนดสีหมวดหมู่ให้กับงานต่างๆ ได้ตามต้องการ
- ✅ **ติดตามงานที่สำเร็จแล้ว:** แยกรายการงานที่ทำเสร็จแล้วเพื่อความสะดวกในการดูประวัติย้อนหลัง
- 💾 **จัดเก็บข้อมูลในเครื่อง (Local Storage):** ข้อมูลทั้งหมดจะถูกบันทึกไว้ในเครื่องของคุณโดยตรง (ออฟไลน์ 100% ไม่ต้องต่อเน็ต ไม่ต้องใช้ฐานข้อมูล)
- 📱 **Responsive Design:** หน้าต่างแอปพลิเคชันรองรับการยืดหยุ่นขนาดหน้าจอ

## 🚀 ดาวน์โหลดและติดตั้ง (Download & Install)

ขณะนี้โปรแกรมอยู่ในเวอร์ชัน **0.1.0 (Prototype)** ซึ่งเป็นตัวทดสอบ คุณสามารถดาวน์โหลดโปรแกรมไปทดลองใช้งานได้ที่หน้า **[Releases](../../releases)** ของ GitHub โดยมีให้เลือก 2 รูปแบบ:

1. **แบบติดตั้ง (Setup / Installer):** ดาวน์โหลดไฟล์ตัวติดตั้งเพื่อติดตั้งลงบนระบบของคุณ
2. **แบบพกพา (Portable):** ดาวน์โหลดไฟล์ Portable สามารถดับเบิ้ลคลิกเปิดใช้งานได้ทันทีโดยไม่ต้องติดตั้ง

## 💻 สำหรับนักพัฒนา (Development)

โปรเจกต์นี้พัฒนาด้วย `HTML`, `CSS`, `JavaScript` (Vanilla) และแพ็กเป็นเดสก์ท็อปแอปพลิเคชันด้วย **Electron**

## 🤖 ผู้ช่วยพัฒนา (AI Assistance)

โปรเจกต์นี้ได้รับการสนับสนุนและช่วยเหลือในด้านการออกแบบโครงสร้าง การเขียนโค้ด และการแก้ปัญหาต่างๆ โดย **AI Assistant** เพื่อช่วยให้การพัฒนาโปรโตไทป์เป็นไปอย่างรวดเร็วและมีประสิทธิภาพมากขึ้น
